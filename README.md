# Light Meter Legal Pages

This is the standalone GitHub Pages package for Light Meter.

It belongs only to the Light Meter app and must not be mixed into another app's legal repository.

## Files

- `index.html`
- `privacy.html`
- `support.html`

## GitHub Repository

Suggested repository:

```text
https://github.com/tongjia323-hub/lightmeter-legal
```

## GitHub Pages URLs

After publishing:

```text
https://tongjia323-hub.github.io/lightmeter-legal/
https://tongjia323-hub.github.io/lightmeter-legal/privacy.html
https://tongjia323-hub.github.io/lightmeter-legal/support.html
```

## App Store Connect

- Privacy Policy URL: `https://tongjia323-hub.github.io/lightmeter-legal/privacy.html`
- Support URL: `https://tongjia323-hub.github.io/lightmeter-legal/support.html`
- Marketing URL: leave empty

## Verification

Run after publishing:

```sh
curl -I -L https://tongjia323-hub.github.io/lightmeter-legal/privacy.html
curl -I -L https://tongjia323-hub.github.io/lightmeter-legal/support.html
```

Both URLs must return `HTTP/2 200` before App Store review submission.
